# BanqueSystem

## Description  
Ce projet est une modélisation d’un système bancaire en C#.  
Il représente les principales entités d’une banque : agences, comptes, clients, cartes et transactions.

---

## Structure du projet  

### Entités principales  
- Banque : gestion des agences et des distributeurs  
- Agence : gestion des comptes bancaires  
- Personne : client pouvant posséder plusieurs comptes et cartes  
- Compte : gestion des opérations (crédit, débit, transfert)  
- Carte : moyen de paiement associé à une personne  
- Transaction : opérations financières (crédit et débit)  

---

## Fonctionnalités  
- Création et suppression de comptes  
- Gestion des clients  
- Crédit, débit et transfert d’argent  
- Gestion des cartes bancaires  
- Génération de relevés de transactions  

---

## Technologies utilisées  
- C#  
- Programmation orientée objet (POO)  

---

## Auteur  
Projet réalisé dans le cadre d’un exercice académique.
